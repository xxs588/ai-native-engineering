# 这一部分先定盘，不先开写

到了这里，项目才正式进入生命周期。先做的不是生成页面和接口，而是把 PRD、范围、技术栈、仓库结构、架构边界和规范资产先立住。

# 这一部分要交付什么

读完这一部分，读者应该能拿着一个真实项目，从需求收敛一路推进到“可以稳定开发”的状态，而不是在混乱上下文里边写边猜。
