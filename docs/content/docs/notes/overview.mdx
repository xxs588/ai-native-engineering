# 这里不是正文分册，而是一个缓冲区

有些内容很有价值，但它的职责不是成为 Part 1 到 Part 5 的正式章节。

比如:

- 为一次分享准备的速览稿。
- 还在演进中的阶段性提纲。
- 某个实验结论、工作流复盘、临时笔记。

这些材料如果硬塞进正文，读起来会很别扭；但如果完全不记录，后面又很容易丢。

所以这里单独开一个 `notes` 分组，专门收纳这类“和全书有关，但不直接属于正文主线”的内容。

:::tip
如果要看覆盖全书主线、适合快速分享的版本，可以直接读 [《分享速览：如果只用一场分享讲 AI Native 工程》](/docs/notes/ai-native-engineering-share)。
:::
